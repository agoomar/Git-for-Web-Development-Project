1. What is Semantic HTML?
	- Semantic HTML describes its meaning to both the developer and the browser (e.g. <form>, <table>) 
2. What is HTML used for?
	-  It's a standard markup language for creating Web pages. It also describes the structure of a Web page. It is the foundation.
3. What is an attribute and where do we put it?
	- An attribute provdies addtional information about elements. It is defined as <a> and specified in the start tag. (e.g. <a href="link here">comment here</a>)
4. What is the h1 tag used for? How many times should I use it on a page?
	- h1 tag is used for important headings and should only be used at least once.
5. Name two tags that have required attributes
	- <img>, <a>, <style>
6. What do we put in the head of our HTML document?
	- <title> tags, links to favicons and metadata. 
7. What is an id?
	- It's an attribute that is used to specify a unique id for an HTML element. 
8. What elements can I add an id to?
	- <h1>, only can use one HTML element within the page. 
9. How many times can I use the same id on a page?
	- Can only use one HTML element within the page.     
10. What is a class?
	- It's an attribute that is used to specify a class for an HTML element    
11. What elements can I add a class to?
	- <div> elements     
12. How many times can I use the same class on a page?
	- Multiple times. HTML elements can share the same class.     
13. How do I get my link to open in a new tab?
	- _blank value    
14. What is the alt attribute in the image tag used for?
	- It specifies an alternat text for an area if the image cannot be displayed or if the user is using a screen reader.     
15. How do I reference an id?
	- # character    
16. What is the difference between a section and a div
	- <div> is a block-level and is often used as a container for other HTML elements.
	- <span> is an inline container used to mark up a part of a text, or a part of a document.    
17. What is CSS used for?
	- CSS is used for describing the presentation of Web pages and to adapt to different types of devices.      
18. How to we select an element? Example - every h2 on the page
	- h2 {}, if we want to select a class we use a period (.) and if we want to select an ID we use a hashtag (#)    
19. What is the difference between a class and an id? - Give me an example of when I might use each one
	- Class is for multiple elements (e.g. <h2> class="city">London</h2>) with the same code repeating but different cities and it will edit those multiple elements the same.
	- As for ID, it is an element with a unique ID and it will only style that unique ID. So mainly used for headers to give it its own unique difference.    
20. How do we select classes in CSS?
	- . (period) is a class selector in CSS    
21. How do we select a p element with a single class of “human””?
	- p.human    
22. What is a parent child selector? When would this be useful?
	- It selects all elements that are a direct child of the specified element.
	- It will be useful to select the lists that you make (e.g. ul li) and styling it on CSS to make it all the same color.     
23. How do you select all links within a div with the class of sidebar?
	- .sidebar    
24. What is a pseudo selector?
	- selector:pseudo-class {}
	- e.g. a:link {insert property: value;}    
25. What do we use the change the spacing between lines?
	- <br>    
26. What do we use to change the spacing between letters?
	- e.g. h1 {letter-spacing: 3px;}    
27. What do we use to to change everything to CAPITALS? lowercase? Capitalize?
	- property of text-transform
	- e.g. div.a {text-transform: uppercase;}    
28. How do I add a 1px border around my div that is dotted and black?
	- div {border: 1px dotted black;}
29. How do I select everything on the page?
	- * selects all elements 
30. How do I write a comment in CSS?
	- /* text */
31. How do I find out what file I am in, when I am using the command line?
	-  $ pwd
32. Using the command line - how do I see a list of files/folders in my current folder?
	- $ ls
33. How do I remove a file via the command line? Why do I have to be careful with this?
	-  $ rm and $rmdir
	- Because it won't give you a prompt and it will delete it right away.
34. Why should I use version control?
	- It's used for teams to help solve problems and to work on the same projects simultaneously.
